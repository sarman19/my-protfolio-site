# Shariar Portfolio (Refactored)

This repository contains a static portfolio website for Shariar (Sarman). The project has been reorganized into a clean, production-ready folder structure while preserving all existing functionality and assets.

## Folder Structure

- `index.html` — landing page (remains at root for GitHub Pages)
- `CNAME` — custom domain configuration (root)

- `pages/` — secondary pages (About, Contact, My Work, Case Study, Blog)
- `admin/` — admin login + dashboard pages
- `css/` — stylesheet files
- `js/` — JavaScript files (including Firebase config)
- `assets/` — static assets
  - `assets/images/` — image assets
  - `assets/icons/` — icon assets (empty placeholder)
  - `assets/fonts/` — custom fonts (empty placeholder)
  - `assets/video/` — video assets (empty placeholder)
- `components/` — shared HTML snippets (navbar, footer)

## Notes

- All internal links and asset references have been updated to match the new directory layout.
- The `blog.html` page was added as a placeholder to avoid broken navigation links.
- `test.html` remains in the root for any development or quick previews.

## Development

Open `index.html` in a browser to view the site locally.
